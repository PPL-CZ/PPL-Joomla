# PPL Plugin pro Joomla
Snadné a efektivní řešení dopravy pro vaše online podnikání

PPL Plugin pro Joomla je praktický nástroj, který umožňuje snadnou integraci přepravních služeb PPL do vašeho e-shopu. Nabídněte svým zákazníkům rychlé a pohodlné doručování zásilek s širokou škálou možností dopravy a sledování.

## Hlavní vlastnosti

### Tvorba štítků
Automatické generování přepravních štítků přímo z administrace Joomla bez nutnosti ručního zadávání údajů.

### Výběr doručovacích metod
Podpora různých typů doručení, jako je doručení na adresu, výdejní místa PPL ParcelShop nebo PPL boxy (ParcelBox, AlzaBox).

### Sledování zásilek (Track & Trace)
Poskytování sledovacího čísla zákazníkům s možností sledování aktuálního stavu objednávky.

### Automatické vyplnění adresy
Integrace s API PPL umožňuje ověřování a automatické doplňování adres zákazníků pro minimalizaci chyb.

### Hromadné zpracování objednávek
Možnost vytvoření štítků a objednání dopravy pro více zásilek najednou.

### Dynamický výpočet ceny dopravy
Kalkulace nákladů na dopravu na základě hmotnosti, rozměrů nebo destinace zásilky.

### Podpora dobírky (COD)
Možnost platby při doručení s automatickým zpracováním částky v objednávkách VirtueMart.

### Notifikace zákazníků
Automatické e-mailové upozornění o stavu zásilky.

### Podpora více skladů
Nastavení různých výchozích adres pro expedici z více skladových míst.

### Jednoduché nastavení
Intuitivní konfigurace přímo v administraci Joomla bez nutnosti složitého nastavování.

## Instalace

1. Stáhněte si plugin z [github repozitáře](../../releases).
2. Přihlaste se do administrace Joomla.
3. Přejděte do **System → Install → Extensions**.
4. Nahrajte stažený soubor `pkg_pplcz_v*.zip`.
5. Klikněte na **Upload & Install**.
6. Přejděte do sekce **Components → PPL CZ** a proveďte potřebná nastavení.

## Požadavky

- **Joomla** 5.0 a vyšší
- **VirtueMart** 4.0 a vyšší
- **PHP** 8.1 a vyšší
- Aktivní účet u PPL

## Obsah balíku

Instalační balík obsahuje:
- **com_pplcz** - Hlavní komponenta pro správu zásilek
- **plg_vmshipment_pplshipping** - VirtueMart plugin pro dopravu PPL
- **plg_vmcustom_pplproduct** - VirtueMart custom plugin pro produkty
